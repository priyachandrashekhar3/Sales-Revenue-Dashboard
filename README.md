# Sales & Revenue Analysis Dashboard

An interactive dashboard project built using Streamlit, Pandas, Matplotlib, and Seaborn for analyzing sales and revenue data.

---

## Features

- KPI Tracking
- Sales Analysis
- Revenue Trend Visualization
- Product Performance Analysis
- Region-wise Sales Insights
- Interactive Filters
- CSV Download Feature

---

## Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib
- Seaborn
