# Sales & Revenue Dashboard + Customer Segmentation

A complete Data Analytics and Machine Learning project developed using Python, Streamlit, Pandas, Matplotlib, Seaborn, and Scikit-learn.

This repository contains:

- Task 1 → Sales & Revenue Analysis Dashboard
- Task 2 → Customer Segmentation using KMeans Clustering

---

# PROJECT OVERVIEW

This project demonstrates:
- Data Analysis
- Business Intelligence
- Dashboard Development
- Machine Learning
- Customer Analytics
- Data Visualization

The project helps businesses analyze:
- Sales performance
- Revenue trends
- Customer behavior
- Customer purchasing patterns
